SubSweep is a subscription and free trial tracker built as a single-page React application. It stores all data in the user's browser localStorage — no backend, no accounts, no tracking. Users can add subscriptions with name, cost, billing cycle, category, start date, and optional free trial end date. The app calculates monthly/yearly totals, shows spending breakdown by category with visual bars, highlights urgent expiring trials with red pulsing badges, and compares total spend to everyday items (coffee, tacos, movie tickets). Built with React 19, Vite, and Tailwind CSS v4. Deployed on Netlify.
